# РусЧат - Мессенджер для россиян

Веб-приложение для общения с функционалом, похожим на Discord.

## Возможности

- ✅ Авторизация только для российских номеров (+7)
- 💬 Текстовые каналы для общения
- 🎤 Голосовые каналы (интерфейс)
- 📈 Просмотр российских акций в реальном времени
- 🖥️ Серверы и каналы
- 👥 Список участников онлайн

## Установка

1. Установите Node.js (если еще не установлен)
2. Установите зависимости:
```
npm install
```

## Запуск

```
npm start
```

Откройте браузер и перейдите по адресу: http://localhost:3000

## Использование

1. Введите имя пользователя
2. Введите российский номер телефона (формат: +7XXXXXXXXXX)
3. Нажмите "Войти"
4. Начните общение в чате
5. Используйте кнопку "📈 Акции" для просмотра котировок российских компаний

## Технологии

- HTML5, CSS3, JavaScript
- Node.js + Express
- WebSocket (ws)

## Примечание

Это демонстрационная версия. Для production использования необходимо добавить:
- Настоящую аутентификацию и базу данных
- Шифрование соединений (HTTPS/WSS)
- API для получения реальных данных по акциям
- Голосовые каналы (WebRTC)
- Загрузку файлов и изображений
